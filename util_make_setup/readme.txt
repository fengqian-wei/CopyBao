学宝打字机(CopyBao) 0.1

该软件是学宝打字机（英文名：CopyBao）0.1公测版。它通过模拟键盘输入，使您能够向虚拟机内复制内容。

打开setup.exe，定位你学宝的解压目录，即可安装。请保留安装程序，以便而后您随时卸载。

它不支持直接输入中文，也不支持文件拖放。如果您有这些需求，可以在虚拟机端安装接收程序（Load Recv），而后使用文件编码功能（Encode File）发送，之后在虚拟机端解码保存。

复制时，请勿在虚拟机内进行其他操作。

限制：
- 目前还不支持Windows XP
- 目前仅对__独享模式__有效。如果您在寝室使用独享模式无法打开学宝自带的虚拟机，不代表本软件不能工作。您仍可试用本软件。
